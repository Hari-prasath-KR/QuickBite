# QuickBite - Campus Catering Management System

## Overview

QuickBite is a full-stack Campus Catering Management System developed using the MERN stack. The platform is designed to simplify food ordering and catering operations within a campus environment by providing dedicated interfaces for Customers, Canteen Managers,Canteen Staff, and Administrators.

The system enables users to browse food items, place orders, make secure online payments, track order status, and manage catering operations efficiently through role-based access control.

## Key Features

### Customer
- User Registration and Login
- Browse Available Food Items
- Add Items to Cart
- Place Orders
- Secure Online Payments using Razorpay
- View Order History
- Track Order Status

### Canteen Manager
- Manage Food Inventory and Stock Levels
- Add, Update, and Remove Menu Items
- Manage Canteen Staff Accounts
- Monitor Canteen Performance through Analytics
- View and Manage Orders
- Track Daily Operations and Sales Metrics

### Delivery Staff
- Interact and Coordinate with Customers
- Verify Payment Status Before Order Handover
- Validate Order Tokens for Offline Order Collection
- Update Order Delivery Status
- Manage Assigned Deliveries
- Update Stock Information During Delivery Operations

### Admin
- Centralized Canteen Management
- User Registration and Account Administration
- Customer, Staff, and Manager Lifecycle Management
- Wallet Management and Financial Oversight
- GST and Tax Configuration Management
- Order and Transaction Monitoring
- Revenue Tracking and Reporting
- Platform Analytics and Performance Monitoring
- System Configuration and Operational Control
- Role-Based Access Management
- Audit and Administrative Governance
